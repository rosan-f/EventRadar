# EventRadar

EventRadar adalah tools CLI berbasis Python untuk mengambil dan menampilkan
informasi seminar, workshop, lomba, dan webinar dari beberapa sumber website.

## Teknologi

- Python
- Requests
- BeautifulSoup

## Design Pattern

- Strategy Pattern
- Factory Pattern

## Rencana Fitur

- Memilih sumber event
- Melakukan scraping informasi event
- Menampilkan event di terminal
- Filter berdasarkan kategori
